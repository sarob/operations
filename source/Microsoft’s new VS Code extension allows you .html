Microsoft’s new VS Code extension allows you to easily create full-stack web applications

by Pradeep

 

@pradeepviswav

Aug 21, 2019 at 14:24 GMT
2 days ago

Microsoft today announced a new Visual Studio Code extension called Web Template Studio. This new extension will allow anyone to easily create full-stack web applications with few clicks. Web Template Studio (WebTS) presents a wizard to quickly create a web application and also offers a ReadMe.md with step by step instructions to start development process.

For now, WebTS supports React, Vue, and Angular for frontend and Node.js and Flask for backend. The web app created using this extension is well-formed, readable code that incorporates cloud services on Azure while implementing proven patterns and best practices.

Template Studio is open source on GitHub, so you can contribute to further improve the experience for other developers.

You can download the WebTS extension for free here.
